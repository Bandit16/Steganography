API Documentation

Encode File

Endpoint: /api/encode_file/

Method: POST

Description: This endpoint allows you to encode a file into an image.

Request

                {
                "Image": "image_file.png",
                "File": "file_to_encode.extension",
                "pin": "1234"
                }

Response

                {
                "ModifiedImage": "http://yourdomain.com/media/modified_image.png"
                }

Decode File

Endpoint: /api/decode_file/

Method: POST

Description: This endpoint allows you to decode a file from an image.

Request

                {
                "Image": "image_with_encoded_file.png",
                "pin": "1234"
                }

Response

                {
                "DecodedFile": "http://yourdomain.com/media/extracted_files/decoded_file.zip"
                }

Encode Message

Endpoint: /api/encode_message/

Method: POST

Description: This endpoint allows you to encode a message into an image.

Request

                {
                "Image": "image_file.png",
                "Message": "Hello World",
                "pin": "1234"
                }

Response

                {
                "ModifiedImage": "http://yourdomain.com/media/modified_image.png"
                }

Decode Message

Endpoint: /api/decode_message/

Method: POST

Description: This endpoint allows you to decode a message from an image.

Request

                {
                "Image": "image_with_encoded_message.png",
                "pin": "1234"
                }

Response

                {
                "DecodedMessage": "Hello World"
                }
